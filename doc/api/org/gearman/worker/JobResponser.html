org.gearman.worker

JobResponser

trait JobResponser extends AnyRef

responser for a job got from gearman server

When a job is fetched from the server side, the worker proccess the job and send the job status to the client through the JobResponser interface

When the job finished, the worker must call one of following methods:

- complete

- fail

- exception
