org.gearman.channel

MessageChannel

trait MessageChannel extends AnyRef

presents a bi-direction communication channel between two peers.

When the client/worker starts, it will try to create a connection to the gearman server. If the communication channel is established successfully, a message channel will be created between the client/worker and gearman server.

The client/worker and the gearman server will get the message channel to present the established communication channel
